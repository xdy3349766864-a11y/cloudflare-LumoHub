# Cloudflare R2 图片管理系统

一个基于 Cloudflare Workers + R2 Storage + KV 的轻量级、高性能图床解决方案。支持 WebP 智能压缩、随机图片接口、防盗链、访问频率限制等功能。

## ✨ 功能特性

- 🚀 **极速部署**：基于 Cloudflare 边缘网络，全球访问低延迟
- 🎨 **精美 UI**：现代化响应式界面，支持拖拽上传
- 🗜️ **智能压缩**：自动将图片转换为 WebP 格式，支持自定义压缩质量
- 🎲 **随机图片接口**：支持 `/random` 接口，可指定目录，支持 JSON/图片流返回
- 🔒 **安全防护**：JWT 登录认证、Referer 防盗链、访问频率限制
- 📱 **移动端适配**：完美适配手机、平板等设备
- 🔧 **无需服务器**：完全基于 Cloudflare 生态，零运维成本

## 📦 部署教程

### 1. 准备工作

你需要一个 Cloudflare 账号（免费版即可）。
可选自定义域名需要域名自己注册
### 2. 创建资源

#### 2.1 创建 R2 存储桶
1. 登录 Cloudflare 控制台，进入 **R2 存储**
2. 点击 **创建存储桶**，输入名称（例如 `my-image-bucket`），点击创建

#### 2.2 创建 KV 命名空间
1. 进入 **Workers 和 Pages** -> **KV**
2. 点击 **创建命名空间**，输入名称（例如 `image-storage-kv`），点击创建

### 3. 部署 Worker

1. 进入 **Workers 和 Pages**，点击 **创建应用程序** -> **创建 Worker**
2. 输入 Worker 名称（例如 `r2-image-storage`），点击 **部署**
3. 部署成功后，点击 **编辑代码**，将本项目 `worker.js` 的代码完整复制粘贴进去，点击 **保存并部署**

### 4. 配置绑定与环境变量

进入你的 Worker -> **设置**，按以下步骤配置：

#### 4.1 绑定（Bindings）
点击 **绑定** -> **添加绑定**：

|   绑定类型   | 变量名 |   选择资源 |
| :--- | :--- | :--- |
| R2 存储桶 | `MY_BUCKET`| 选择你刚创建的 R2 存储桶 |
| KV 命名空间 | `KV` | 选择你刚创建的 KV 命名空间 |

#### 4.2 环境变量
点击 **环境变量** -> **添加变量**：

**必须配置：**

|   变量名   |         示例值       | 说明 |
| :--- | :--- | :--- |
| `R2_ADMIN_USER` | `your_username` | 后台登录用户名 |
| `R2_ADMIN_PASS` | `StrongPass123!` | 后台登录密码（建议强密码） |
| `JWT_SECRET` | `32位以上随机字符串` | JWT 签名密钥（务必复杂） |

**可选配置（按需填写）：**

| 变量名 | 默认值 | 说明 |
| :--- | :--- | :--- |
| `CORS_ALLOW_ORIGIN` | `*` | 跨域允许来源，生产环境建议填域名 |
| `ALLOWED_REFERERS` | 空 | 防盗链白名单，多域名用英文逗号分隔 |
| `RATE_LIMIT_MAX` | `20` | 频率限制：单 IP 时间窗口内最大请求数 |
| `RATE_LIMIT_WINDOW` | `60` | 频率限制：时间窗口（秒） |

配置完成后，**重新部署一次代码**使配置生效。

## 🎯 使用说明

### 1. 登录后台
访问你的 Worker 域名，使用配置的 `R2_ADMIN_USER` 和 `R2_ADMIN_PASS` 登录。

### 2. 上传图片
- 点击或拖拽图片到上传区域
- 可在右侧调整 WebP 压缩质量
- 上传成功后会自动生成链接

### 3. 随机图片接口

**基础用法：**
https://your-worker.domain/random
